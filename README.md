# MINERIO FLOW 
Interface visual do sistema de manitoramentos de minerios, por meio dele o usuario podera visualizar e atualizar o status do lote de minerio, registra novos lotes etc.

<i>Para melhor compreensão da logica do negicio, visite o repositorio remoto do <b>GitHub:</b></i> <a href="@" >Link do repositorio do back-end</a>

## GET START
### GET REPOSITORY
```bash
git clone https://github.com/Lukas-Souza/MINERIO-FLOW_FRONT-END.git
```
### CD REPOSITORY
```bash
cd MINERIO-FLOW_FRONT-END
```
### RUN PROJECT
```bash
npm run dev
```
